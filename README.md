# Cybershi

Каркас динамичного 2.5D-шутера на Unity: вид сбоку, мувсет в духе V1 из ULTRAKILL
(рывок, подкат, высокий прыжок, прыжок от стен, граунд-слэм), буллет-хелл как в Touhou,
3 вида оружия, динамичная музыка, система звуков, смена перспективы вплоть до перехода в 3D,
**новая Input System**, главное меню с настройками, игровой HUD, пауза и **волны врагов с боссом по триггеру**.
Вся графика — на плейсхолдерах, легко заменяемых на спрайты.

## Как запустить за 4 шага
1. Unity **6 (6000.0+)** рекомендуется (используется `Rigidbody.linearVelocity`; на 2022 замените на `velocity`).
2. Package Manager → установите **Input System**; Project Settings → Player → **Active Input Handling = Both**
   (Unity попросит перезапуск — согласитесь).
3. Скопируйте папку `Assets/` в Unity-проект.
4. Меню Unity: **Cybershi → Build Game (Menu + Arena)** → откроется `MainMenu` → **Play** → «Играть».

Управление: **WASD** — движение, **мышь** — прицел, **ЛКМ** — огонь, **Shift** — рывок,
**Ctrl** — подкат/слэм, **Space** — прыжок, **1/2/3** — оружие, **Esc** — пауза. Геймпад поддерживается.
