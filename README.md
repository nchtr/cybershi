# Cybershi

Каркас динамичного 2.5D-шутера на Unity: вид сбоку, мувсет в духе V1 из ULTRAKILL
(рывок, подкат, высокий прыжок, прыжок от стен, граунд-слэм), буллет-хелл как в Touhou,
3 вида оружия, динамичная музыка, система звуков, смена перспективы вплоть до перехода в 3D.
Вся графика — на плейсхолдерах, легко заменяемых на спрайты.

## Как запустить за 3 шага
1. Скопируйте папку `Assets/` в Unity-проект (Unity **2021.3 LTS+**).
2. Project Settings → Player → **Active Input Handling = Input Manager (Old)** (или Both).
3. Меню Unity: **Cybershi → Build Sample Scene** → откроется `Assets/Scenes/SampleArena.unity` → **Play**.

Управление: **WASD** — движение, **мышь** — прицел, **ЛКМ** — огонь, **Shift** — рывок,
**Ctrl** — подкат/слэм, **Space** — прыжок, **1/2/3** — оружие.
