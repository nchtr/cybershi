# Cybershi

Готовый проект динамичного 2.5D-шутера на Unity: вид сбоку, мувсет в духе V1 из ULTRAKILL
(рывок-парирование, постоянный подкат, прыжок от любой стены ×3, граунд-слэм), буллет-хелл как в Touhou
с **грейзом** и **парированием** подсвеченных пуль, **система стиля и комбо** (D…SS, свежесть оружия),
**вампиризм в упор** с анти-абьюзом, **хит-стоп**, **6 оружий** с альт-огнём (хитскан-револьвер + цепочки
монеток с рикошетом/сплитом, дробовик с накачкой, гвоздомёт с магнитом, ракетка с мячиком,
**рельсотрон** с пробитием, **ракетница** с рокет-джампом), **5 типов врагов** на конструкторной базе
(дрон, раннер, снайпер с лазером, гибрид с перекатами, огнемётчик), волны с боссом по триггеру,
**переход между уровнями**, динамичная музыка + музыка меню, смена перспективы вплоть до 3D,
новая Input System, меню/настройки/HUD/пауза. Вся графика — плейсхолдеры, легко заменяемые на спрайты.

## Как запустить
1. Unity **6 (6000.0+)** (используется `Rigidbody.linearVelocity`; на 2022 замените на `velocity`).
2. Package Manager → установите **Input System**; Project Settings → Player → **Active Input Handling = Both**
   (без пакета тоже заработает — есть фолбэк на старый ввод).
3. Откройте `Assets/Scenes/MainMenu.unity` → **Play** → «Играть» (сцены готовы и прописаны в Build Settings).

Управление: **WASD** — движение, **мышь** — прицел, **ЛКМ** — огонь, **ПКМ** — альт-огонь,
**Shift** — рывок (парирует), **Ctrl** — подкат/слэм, **Space** — прыжок, **1-6** — оружие,
**Esc** — пауза. Геймпад поддерживается.
